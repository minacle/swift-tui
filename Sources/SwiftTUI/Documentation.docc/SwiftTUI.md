# ``SwiftTUI``

Build terminal user interfaces with a SwiftUI-like API.

## Overview

The `SwiftTUI` module is the convenience entry point for the package. It
re-exports the complete application foundation from `SwiftTUIEssentials` and
the standard controls from `SwiftTUIControls`.

Import one of the narrower modules directly when you want an explicit
dependency boundary, or import `SwiftTUI` to use the complete library.

Browse the [Essentials API](../swifttuiessentials/) and the
[Controls API](../swifttuicontrols/) for the declarations owned by each
module.
